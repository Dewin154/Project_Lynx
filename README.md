## About the Project
**Project Lynx** is a **2D pixel-art platformer** with **Metroidvania elements**, developed using **Godot Engine**. It serves as a successor to *Samurai's Nightmare*.

## Features
- **Classic Metroidvania Gameplay**
- **Pixel-Art Graphics**
- **Godot Engine**
